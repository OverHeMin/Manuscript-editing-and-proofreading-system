# 前端交互、工作台与微信小程序

## Web 端主导航

- 工作台
- 稿件
- 审稿
- 编加
- 校对
- PDF 核对
- 知识库
- 模板治理
- 学习审核
- 审计
- 系统设置

## 角色工作台

`admin`：系统状态、待发布模板、失败任务、高风险告警

`knowledge_reviewer`：待审核知识、学习候选、相似候选聚类

`screener / editor / proofreader`：待处理稿件、最近结果、风险提示、人工复核待办

`user`：我的稿件、状态、下载记录

## 关键流程

- 上传稿件并勾选类型
- AI 识别类型并人工确认
- 独立发起审稿 / 编加 / 校对
- ONLYOFFICE 预览与批注评估
- 校对人工确认
- 知识审核
- 学习候选审核
- 模板发布

## 微信小程序

Phase 2 优先承接 `knowledge` 模块：

- 待审核知识列表
- 知识详情
- 通过 / 驳回
- 审核备注
- 已审核记录查询

不承接全文处理与复杂文档交互。
