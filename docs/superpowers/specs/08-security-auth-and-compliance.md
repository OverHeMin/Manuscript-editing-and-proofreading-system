# 认证、安全与合规

## 认证

- 本地账号密码起步
- 预留后续 LDAP / SSO 扩展位
- 密码使用 `Argon2id` 或 `bcrypt`
- 登录失败限制
- 会话超时与刷新机制

## 敏感操作保护

以下操作应做二次确认或高等级审计：

- 发布模板
- 批量审批知识或学习候选
- 删除稿件
- 导出敏感记录
- 调整系统配置

## 密钥与配置安全

- 密钥不进代码库
- 环境变量或密钥管理系统保存
- 不同环境使用不同密钥

## 脱敏与数据边界

- 学习链路强制脱敏预检
- 定义哪些数据禁止进入学习层
- 定义哪些内容只能查看脱敏版本

## 审计

必须审计：

- 登录 / 登出 / 密码重置
- 权限变更
- 模板发布
- 知识审核
- 学习候选审批
- 敏感数据导出
